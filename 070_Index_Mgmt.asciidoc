== 索引管理

我们已经看到 Elasticsearch 让开发一个新的应用变得简单，不需要任何预先计划或设置。
不过，要不了多久你就会开始想要优化索引和搜索过程，以便更好地适合您的特定用例。
这些定制几乎围绕着索引和类型的方方面面，在本章，我们将介绍管理索引和类型映射的 API 以及一些最重要的设置。
